Comment installer le jeu Sims 4 Original ?

Les Sims... Te souviens-tu ? Ce jeu de ton enfance où imagination et délire se rencontraient pour former un tout artistique. Nous avons traversé plusieurs générations, des premiers Sims avec un gameplay novateur, aux Sims 2 avec leurs interactions farfelues qui ont conquis plus d'un cœur, jusqu'aux Sims 3 nous offrant un univers ouvert regorgeant de caractéristiques innovantes et inspirantes. Et maintenant, nous avons les Sims 4.

Pour votre plus grand plaisir, ils sont totalement GRATUITS. C'est enfin arrivé ! Les Sims 4 sont disponibles gratuitement sur les plateformes suivantes : Sony Playstation, PC, MAC, Microsoft Xbox, mais malheureusement, pas sur Nintendo Switch.

Maintenant passons à l'essentiel : le téléchargement et l'installation ! Pour cet opus, nous vous avons facilité la tâche. Voyez-vous, notre site est totalement dédié au téléchargement, à l'installation et à la manipulation du jeu.

Pour commencer, il faut télécharger le jeu Les Sims 4. Pour ce faire, vous pouvez cliquer sur ce lien qui vous redirigera vers notre page Packs pour effectuer le(s) téléchargement(s) adéquat(s).

Cliquez sur le lien de téléchargement et vous serez redirigé sur le site officiel d'Electronic Arts

Une fois arrivé sur le site d'Electronic Arts, vous pourrez apercevoir le lien de téléchargement "Télécharger maintenant". Survolez ce lien et vous verrez une liste d'éléments permettant de télécharger le jeu selon le type de plateforme que vous utilisez.

Faites bien attention de choisir le lien qui correspond à votre système d'exploitation. Par exemple, si vous êtes sur Windows, vous pouvez choisir EA app pour Windows ou si vous préférez garder votre jeu dans une bibliothèque différente de EA, choisissez entre Steam et Epic Games. Dans notre cas, nous allons choisir EA app pour Windows.

Allez maintenant dans vos téléchargements et lancez le programme que vous venez de télécharger.

Installez l'application EA. Vous n'aurez pas à patienter longtemps, l'installation de l'application se fait rapidement. Tout dépendra de la vitesse de votre connexion.


Vous pouvez désormais ouvrir le logiciel que vous venez d'installer et vous y connecter. Pour la connexion, vous avez un panel de choix à votre disposition. Choisissez celui qui vous conviendra le mieux et qui vous permettra de vous connecter rapidement lors de vos prochaines connexions. Ne patientez pas à se connecter rapidement.


Dans la barre de recherche, tapez Les Sims 4. Vous trouverez rapidement le jeu dans la bibliothèque EA en téléchargement gratuit. Lancez le téléchargement et patientez un peu. Dès lors que le jeu sera installé sur votre PC, vous le retrouverez dans votre bibliothèque. Si vous avez pris/acheté des extensions, vous verrez la mention (Nombre) Extension comme sur la photo ci-jointe.

Mes félicitations, vous avez fini l'installation du jeu Les Sims 4 Original. Vous pouvez maintenant profiter du jeu sans interruption !
